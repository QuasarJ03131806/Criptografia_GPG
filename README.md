# Criptografia_GPG
Repositório da Disciplina: Conceitos de Segurança da Informação

Este repositório é dedicado ao conteúdo das aulas da disciplina Conceitos de Segurança da Informação. Aqui você encontrará materiais e recursos que abrangem desde os princípios fundamentais — como confidencialidade, integridade e disponibilidade — até tópicos avançados voltados à proteção de dados e comunicação segura.

Entre os temas abordados, destacam-se:

Criptografia assimétrica, com foco no algoritmo RSA, amplamente utilizado para troca segura de chaves e autenticação;
Chaves de segurança com diferentes tamanhos de bits, como 2048 ou 4096 bits, essenciais para garantir robustez contra ataques de força bruta;
Ferramentas de criptografia como PGP (Pretty Good Privacy) e GPG (GNU Privacy Guard), empregadas para assinar, criptografar e proteger e-mails e arquivos;
Soluções de criptografia de disco, incluindo:
BitLocker (Windows), para proteção de volumes inteiros;
VeraCrypt e VileVault (macOS), utilizados para criptografia de arquivos e discos;
LUKS (Linux Unified Key Setup), amplamente adotado no Linux para criptografar partições, geralmente integrado ao dm-crypt;
GnuPG (GPG), como principal solução de criptografia e assinatura de arquivos e mensagens no Linux.
